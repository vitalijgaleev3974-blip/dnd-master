# 🐉 DnD Master — AI Dungeon Master для Android

Полнофункциональный AI-Мастер Подземелий для D&D 5e (2024) на Android.

## 📋 Возможности

- **Создание персонажей** — все расы, классы, предыстории D&D 5e 2024
- **AI-Мастер** — выбор между Claude (Anthropic) или Gemini (Google)
- **Лист персонажа** — HP, характеристики, навыки, заклинания, инвентарь
- **Квест-трекер** — квесты автоматически обновляются AI-Мастером
- **Кубики** — d4, d6, d8, d10, d12, d20, d100 + произвольная формула (2d6+3)
- **Боевая система** — инициатива, раунды, отслеживание состояний
- **Автообновление** — Мастер сам обновляет HP, инвентарь, квесты через JSON
- **До 6 игроков** одновременно
- **Тёмная фэнтези тема**

## 🚀 Установка

### Требования
- Android Studio Hedgehog или новее
- JDK 11+
- Android SDK 29+ (Android 10)

### Шаги

1. **Открой проект** в Android Studio:
   ```
   File → Open → выбери папку dnd_master
   ```

2. **Синхронизируй Gradle**:
   ```
   File → Sync Project with Gradle Files
   ```

3. **Запусти** на устройстве или эмуляторе

4. **Получи API ключ**:
   - **Claude**: https://console.anthropic.com → API Keys
   - **Gemini**: https://aistudio.google.com → Get API Key

5. **Введи ключ** в приложении при создании сессии или в Настройках

## 🎮 Как играть

1. Запусти приложение → **«+»** — новое приключение
2. Выбери ИИ-Мастера и введи API ключ
3. Нажми на персонаж-чип → **«+ Персонаж»** для каждого игрока
4. Заполни данные персонажа (раса, класс, характеристики)
5. Пиши действия от лица персонажа — AI-Мастер ответит!
6. 🎲 — бросок кубиков в любой момент
7. Значок персонажа — просмотр листа персонажа

## 🤖 Как работает AI-Мастер

AI получает:
- Текущую сцену и состояние мира
- Список всех персонажей с HP, состояниями
- Историю последних 20 сообщений
- Действие игрока

AI отвечает:
- Нарративом истории
- Опциональным JSON-блоком `game_state_update` для обновления:
  - XP, HP, инвентарь, золото
  - Состояния и эффекты
  - Квесты (добавить/выполнить/обновить)
  - Начало/конец боя
